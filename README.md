Installation
------------
#### Install package
Run command
```
composer require yiimaker/newsletter
```
or add
```json
"yiimaker/newsletter": "dev-master"
```
to the require section of your composer.json.

#### Apply migrations
```
./yii migrate --migrationPath=@vendor/yiimaker/yii2-newsletter/src/migrations
```
